## About the Game
Pop Sniper Op is a fun bubble shooting game made by Ophir. 
Your goal is to shoot the bubbles and delete them all to win!

### How to play on Mobile:
1. Open the link above.
2. Tap the screen to shoot.
3. Add to your Home Screen to play like a real app!
